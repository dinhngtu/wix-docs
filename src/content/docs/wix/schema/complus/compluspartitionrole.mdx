Defines a COM+ partition role. Partition roles can not be
 created; this element can only be used as a locater to reference an existing
 role.


## Attributes

<BlockWrapper>

  <BlockReference
    content={{
      title: "Id",

    }}
  >
  Identifier for the element.
  </BlockReference>

  <BlockReference
    content={{
      title: "Name",
      type: "string",
      required: true
    }}
  >
  Name of the partition role.
  </BlockReference>

  <BlockReference
    content={{
      title: "Partition",
      type: "string"
    }}
  >
  The id of a ComPlusPartition element representing the partition
   the role belongs to.
  </BlockReference>

</BlockWrapper>
