Property value for a Package or Module.


## Attributes

<BlockWrapper>

  <BlockReference
    content={{
      title: "Admin",
      type: "yesno"
    }}
  >
  Denotes that the property is saved during [administrative installations](https://learn.microsoft.com/en-us/windows/win32/msi/administrative-installation).
   Adds the property name to the [AdminProperties property](https://learn.microsoft.com/en-us/windows/win32/msi/adminproperties).
  </BlockReference>

  <BlockReference
    content={{
      title: "ComplianceCheck",
      type: "yesno"
    }}
  >
  Adds a row to the CCPSearch table. This attribute is only valid when this Property contains a search element.
  </BlockReference>

  <BlockReference
    content={{
      title: "Hidden",
      type: "yesno"
    }}
  >
  Denotes that the Property is not logged during installation. See the [MsiHiddenProperties Property](https://learn.microsoft.com/en-us/windows/win32/msi/msihiddenproperties) for more information.
  </BlockReference>

  <BlockReference
    content={{
      title: "Id",
      type: "string",
      required: true
    }}
  >
  Unique identifier for Property.
  </BlockReference>

  <BlockReference
    content={{
      title: "Secure",
      type: "yesno"
    }}
  >
  Denotes that the Property can be passed to the server side when doing a managed installation with elevated privileges. See the [SecureCustomProperties Property](https://learn.microsoft.com/en-us/windows/win32/msi/securecustomproperties) for more information.
  </BlockReference>

  <BlockReference
    content={{
      title: "SuppressModularization",
      type: "yesno"
    }}
  >
  Use to suppress modularization of this property identifier in merge modules.
   Using this functionality is strongly discouraged; it should only be
   necessary as a workaround of last resort in rare scenarios.
  </BlockReference>

  <BlockReference
    content={{
      title: "Value",
      type: "string"
    }}
  >
  Sets a default value for the property. The value will be overwritten if the Property is used for a search.
  </BlockReference>

</BlockWrapper>
