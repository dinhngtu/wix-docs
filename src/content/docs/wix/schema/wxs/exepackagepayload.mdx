Describes information about the ExePackage payload.
 Cannot be specified if the owning ExePackage specified any of SourceFile, Name, DownloadUrl, or Compressed.


## Attributes

<BlockWrapper>

  <BlockReference
    content={{
      title: "CertificatePublicKey",
      type: "string"
    }}
  >
  Optional public key of the certificate used to sign the payload. It is not recommended to use this attribute and rely on the Hash alone.
  </BlockReference>

  <BlockReference
    content={{
      title: "CertificateThumbprint",
      type: "string"
    }}
  >
  Optional thumbprint of the certificate used to sign the payload. It is not recommended to use this attribute and rely on the Hash alone.
  </BlockReference>

  <BlockReference
    content={{
      title: "Compressed",
      type: "YesNoDefaultTypeUnion",
      typeLink: "../yesnodefaulttype/",
      typeTitle: "Values of this type will either be \"default\", \"yes\", or \"no\"."
    }}
  >
  Whether the package payload should be embedded in a container or left as an external payload.
  </BlockReference>

  <BlockReference
    content={{
      title: "Description",
      type: "string"
    }}
  >
  Description of the file from version resources.
   If Hash is not specified, must not be specified.
  </BlockReference>

  <BlockReference
    content={{
      title: "DownloadUrl",
      type: "string"
    }}
  >
  The URL to use to download the package. The following substitutions are supported:
  
  
  - \{0} is replaced by the package Id.
  - \{1} is replaced by the payload Id.
  - \{2} is replaced by the payload file name.
  
  </BlockReference>

  <BlockReference
    content={{
      title: "Hash",
      type: "HexType",
      typeLink: "../hextype/",
      typeTitle: "This type supports any hexadecimal number. Both upper and lower case are supported for letters appearing in the number. This type also includes the empty string: \"\"."
    }}
  >
  SHA-512 hash of the RemotePayload.
   If SourceFile is specified, must not be specified.
   If specified then Name, DownloadUrl, and Size are required.
  </BlockReference>

  <BlockReference
    content={{
      title: "Id",
      type: "string"
    }}
  >
  The identifier of the package payload element.
  </BlockReference>

  <BlockReference
    content={{
      title: "Name",
      type: "string"
    }}
  >
  The destination path and file name for this chain payload.
   Use this attribute to rename the chain entry point or extract it into a subfolder.
   The default value is the file name from the SourceFile attribute.
   At a minimum, the Name or SourceFile attribute must be specified.
   This must be a relative path ('\foo' or 'C:\foo' is not allowed).
  </BlockReference>

  <BlockReference
    content={{
      title: "ProductName",
      type: "string"
    }}
  >
  Product name of the file from version resouces.
   If Hash is not specified, must not be specified.
  </BlockReference>

  <BlockReference
    content={{
      title: "Size",
      type: "Integer",
      typeLink: "../integer/",
      typeTitle: "Values of this type must be a non-negative integer or a preprocessor variable with the format `$(Variable)` or `$(var.Variable)`."
    }}
  >
  Size of the remote file in bytes.
   Required if Hash is specified, otherwise must not be specified.
  </BlockReference>

  <BlockReference
    content={{
      title: "SourceFile",
      type: "string"
    }}
  >
  Location of the package to add to the bundle.
   The default value is the Name attribute, if provided.
   If Hash is specified, must not be specified.
  </BlockReference>

  <BlockReference
    content={{
      title: "Version",
      type: "WixVersionType",
      typeLink: "../wixversiontype/",
      typeTitle: "Values of this type will be a 1-part, 2-part, 3-part or 4-part version number or a semantic version."
    }}
  >
  Version of the remote file.
   If Hash is not specified, must not be specified.
  </BlockReference>

</BlockWrapper>
