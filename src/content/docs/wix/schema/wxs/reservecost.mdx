Disk cost to reserve in a folder for running locally and/or from source.


## Attributes

<BlockWrapper>

  <BlockReference
    content={{
      title: "Directory",
      type: "string"
    }}
  >
  Adds the amount of disk space specified in RunFromSource or RunLocal to the volume cost of the device containing the directory.
   If this attribute is not set, it will default to the directory of parent component.
  </BlockReference>

  <BlockReference
    content={{
      title: "Id",
      type: "string",
      required: true
    }}
  >
  A primary key that uniquely identifies this ReserveCost entry.
  </BlockReference>

  <BlockReference
    content={{
      title: "RunFromSource",
      type: "Integer",
      typeLink: "../integer",
      typeTitle: "Values of this type must be a non-negative integer or a preprocessor variable with the format `$(Variable)` or `$(var.Variable)`.",
      required: true
    }}
  >
  The number of bytes of disk space to reserve if the component is installed to run from source.
  </BlockReference>

  <BlockReference
    content={{
      title: "RunLocal",
      type: "Integer",
      typeLink: "../integer",
      typeTitle: "Values of this type must be a non-negative integer or a preprocessor variable with the format `$(Variable)` or `$(var.Variable)`.",
      required: true
    }}
  >
  The number of bytes of disk space to reserve if the component is installed to run locally.
  </BlockReference>

  <BlockReference
    content={{
      title: "Subdirectory",
      type: "string"
    }}
  >
  This attribute defines one or more directories below the directory referenced by the Directory attribute
   to add cost to.
  </BlockReference>

</BlockWrapper>
