Use to sequence a custom action.


## Attributes

<BlockWrapper>

  <BlockReference
    content={{
      title: "Action",
      type: "string",
      required: true
    }}
  >
  The CustomAction to which the Custom element applies.
  </BlockReference>

  <BlockReference
    content={{
      title: "After",
      type: "string"
    }}
  >
  The name of the standard or custom action after which this action should be performed. Mutually exclusive with Before, OnExit, and Sequence attributes
  </BlockReference>

  <BlockReference
    content={{
      title: "Before",
      type: "string"
    }}
  >
  The name of the standard or custom action before which this action should be performed. Mutually exclusive with OnExit, After, and Sequence attributes
  </BlockReference>

  <BlockReference
    content={{
      title: "Condition",
      type: "string"
    }}
  >
  Optional condition that determines whether the action should be executed.
  </BlockReference>

  <BlockReference
    content={{
      title: "OnExit",
      type: "ExitType",
      typeLink: "../exittype",
      typeTitle: "Value indicates that this action is executed if the installer returns the associated exit type. Each exit type can be used with no more than one action.\r\n Multiple actions can have exit types assigned, but every action and exit type must be different. Exit types are typically used with dialog boxes."
    }}
  >
  Mutually exclusive with Before, After, and Sequence attributes This attribute's value must be one of the following:
  - *success*
  - *cancel*
  - *error*
  - *suspend*
  </BlockReference>

  <BlockReference
    content={{
      title: "Overridable",
      type: "yesno"
    }}
  >
  If "yes", the sequencing of this action may be overridden by sequencing elsewhere.
  </BlockReference>

  <BlockReference
    content={{
      title: "Sequence",
      type: "Integer",
      typeLink: "../integer",
      typeTitle: "Values of this type must be a non-negative integer or a preprocessor variable with the format `$(Variable)` or `$(var.Variable)`."
    }}
  >
  The sequence number for this action. It is recommended to use one of the other mutually exclusive attributes: OnExit, After, and Before.
  </BlockReference>

</BlockWrapper>
