Create a reference to a Directory element in another Fragment.


## Attributes

<BlockWrapper>

  <BlockReference
    content={{
      title: "DiskId",
      type: "DiskIdType",
      typeLink: "../diskidtype/",
      typeTitle: "Values of this type must be an integer or the value of one or more preprocessor variables with the format `$(Variable)` or `$(var.Variable)`."
    }}
  >
  Sets the default disk identifier for the files contained in this directory.
   This attribute's value may be overridden by a child Component, Directory,
   Merge or File element. See the File or Merge elements' DiskId attribute for
   more information.
  </BlockReference>

  <BlockReference
    content={{
      title: "FileSource",
      type: "string"
    }}
  >
  Used to set the file system source for this DirectoryRef's child elements.
   For more information, see [Specifying source files](../../../tools/payloads/).
  </BlockReference>

  <BlockReference
    content={{
      title: "Id",
      type: "string",
      required: true
    }}
  >
  The identifier of the Directory element to reference.
  </BlockReference>

</BlockWrapper>
