---
title: "KeepRemoveForceType type"
---

Values of this type will either be "force", "keep", or "remove".

## Enumeration values

  - *force*: Always cache the package during Cache, Install, Modify, Repair, and Layout.
  - *keep*: Keep the package cached when the package is requested to be present.
  - *remove*: Always uncache the package after Cache, Install, Modify, Repair, Uninstall, and Layout.
