Uses WixManagedBootstrapperApplicationHost as the Bootstrapper Application for a Bundle.

 _Available in WiX v4. Deprecated in WiX v5; use WixPrerequisiteBootstrapperApplication instead._


## Attributes

<BlockWrapper>

  <BlockReference
    content={{
      title: "AlwaysInstallPrereqs",
      type: "string"
    }}
  >
  When not set or set to "no", the prereq BA is only used when the .NET Framework runtime can't be loaded.
   When set to "yes", the prereq BA will always run before the .NET Framework runtime is attempted to be loaded.
   If set to "yes" and all prereq packages are already installed, then the prereq BA's UI is not shown.
  </BlockReference>

  <BlockReference
    content={{
      title: "LocalizationFile",
      type: "string"
    }}
  >
  Source file of the theme localization .wxl file.
  </BlockReference>

  <BlockReference
    content={{
      title: "LogoFile",
      type: "string"
    }}
  >
  Source file of the logo graphic.
  </BlockReference>

  <BlockReference
    content={{
      title: "Theme",
      type: ""
    }}
  >
  The built-in theme to use. The default is "standard". This attribute's value must be one of the following:
  - *none*
  - *standard*
  </BlockReference>

  <BlockReference
    content={{
      title: "ThemeFile",
      type: "string"
    }}
  >
  Source file of the theme XML.
  </BlockReference>

</BlockWrapper>
