Harvest files from a folder.


## Attributes

<BlockWrapper>

  <BlockReference
    content={{
      title: "Id",
      type: "string"
    }}
  >
  Unique identifier for this harvest configuration. This value will generated if not specified.
  </BlockReference>

  <BlockReference
    content={{
      title: "SelfReg",
      type: "wxs:YesNoTypeUnion"
    }}
  >
  Capture self-registration for executable files during harvesting. Self-reg is only attempted when the attribute is set to 'yes'. The default is 'no'.
  </BlockReference>

  <BlockReference
    content={{
      title: "SelfRegExecutableArguments",
      type: "string"
    }}
  >
  Command-line arguments to pass to an executable when capturing self-registration. Ignored unless SelfReg attribute is 'yes'.
  </BlockReference>

  <BlockReference
    content={{
      title: "SelfRegExtensions",
      type: "string"
    }}
  >
  Whitespace separated list of file extensions to capture self-registration. Ignored unless SelfReg attribute is 'yes'. Default set of extensions is: .exe, .com, .dll, .ocx, .tlb, .olb
  </BlockReference>

  <BlockReference
    content={{
      title: "SelfRegWarnOnError",
      type: "wxs:YesNoTypeUnion"
    }}
  >
  Downgrade error messages encountered during self-registration capture to warnings. Ignored unless SelfReg attribute is 'yes'. The default is 'no'.
  </BlockReference>

  <BlockReference
    content={{
      title: "Source",
      type: "string",
      required: true
    }}
  >
  Root of folder to harvest relative to the harvesting document. If no wildcards are specified all files under the root are harvested. If any wildcard
   is specified only files matching the wildcard are included. For example, "content\*.dat" will include all .dat files from the "content" folder beside
   the .wxs file.
  </BlockReference>

  <BlockReference
    content={{
      title: "Strategy",
      type: "HarvestStrategyTypeUnion",
      typeLink: "../harveststrategytype/",
      typeTitle: "Values of this type will either be \"default\", \"singleFileComponents\" or \"contentFileComponentsWithGeneratedKeyPath\"."
    }}
  >
  Harvest strategy to apply.
  </BlockReference>

  <BlockReference
    content={{
      title: "SuppressFont",
      type: "wxs:YesNoTypeUnion"
    }}
  >
  Prevent automatically marking ".ttf" files as a font. The default is 'no'.
  </BlockReference>

</BlockWrapper>
